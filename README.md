Teterly | CEITI – Extensie Chrome de productivitate
==========================================

Descriere
--------
Teterly este o extensie Chrome care îți transformă noul tab într-un panou de productivitate. Ai într-un singur loc lista de taskuri, timer pentru focus, calendar zilnic.

Funcționalități
---------------
- To Do List pentru organizarea taskurilor zilnice
- Timer pentru focus (tip Pomodoro sau clasic)
- Calendar simplu pentru vizualizarea zilei

Tehnologii folosite
-------------------
- HTML, CSS, JavaScript
- Chrome Extensions API

Structura proiectului
---------------------
- manifest.json           fișierul de configurare al extensiei
- popup.html              interfața principală a extensiei
- popup.css               stilurile pentru interfață
- popup.js                logica principală a extensiei
- icons/                  iconițele extensiei
- assets/                 imagini și resurse suplimentare
- README.md               documentația proiectului

Instalare locală
----------------
1. Descarcă sau clonează repository-ul:
   git clone https://github.com/hodorojasorin/Teterly.git

2. Deschide Google Chrome.

3. Intră în:
   chrome://extensions/

4. Activează Developer Mode.

5. Apasă pe Load unpacked.

6. Selectează folderul Teterly din calculator.

7. Deschide un New Tab și verifică interfața Teterly.

Workflow de dezvoltare
----------------------
1. Faci modificări în fișierele HTML, CSS, JS.
2. Revii în chrome://extensions/.
3. Apeși pe Reload la extensia Teterly.
4. Deschizi un New Tab și verifici schimbările.
